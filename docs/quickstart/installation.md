# Installation

We recommend that you use Seam's client libraries in the language of your choice.

- Javascript / Typescript
- Python
- Ruby

## Install Library

You can use your favorite package manager to install Seam's client libraries.

{% tabs %}
{% tab title="Javascript" %}

```console
$ npm i seamapi
```

{% endtab %}

{% tab title="Python" %}

```console
$ pip install seamapi
```

{% endtab %}

{% tab title="Ruby" %}

```console
$ bundle add seamapi
```

{% endtab %}
{% endtabs %}
